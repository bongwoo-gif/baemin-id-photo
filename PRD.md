# PRD: 사원증 사진 제출 웹앱 v1.0 (MVP)

## 목적
신규 입사자가 외부 IP에서 접속하여 사원증용 사진을 직접 업로드·크롭하고 로컬에 저장할 수 있는 웹앱

## 범위 (MVP - 1단계)
- 저장/전송 기능 없음 (로컬 다운로드만)
- 단일 HTML 파일로 구성
- 외부 라이브러리는 CDN으로 처리

## 기능 요구사항

### 1. 사진 업로드
- 파일 선택 버튼 (jpg, png 허용)
- 모바일 카메라 직접 촬영 지원

### 2. 얼굴 자동 인식 및 배치
- MediaPipe Face Detection 사용
- 얼굴 감지 후 원형 프레임 안에 자동 배치
- 얼굴 기준 상하좌우 여백 10% 자동 적용
- 감지 실패 시 수동 조절 모드로 전환

### 3. 수동 조절 UI
- 이미지 드래그로 위치 조절
- 핀치 또는 슬라이더로 크기 조절

### 4. 사원증 프레임 오버레이
- 원형 크롭 영역 표시
- 실제 사원증 비율 (85.6mm x 54mm, CR-80 기준) 가이드

### 5. 완성 및 저장
- "저장" 버튼 클릭 시 크롭된 이미지 로컬 다운로드
- 파일명: 이름_입사일.jpg 형식
- 이름, 입사일 입력 필드 포함

## 기술 스택
- 단일 HTML 파일 (index.html)
- MediaPipe Face Detection (CDN)
- Canvas API (크롭 처리)
- 순수 JavaScript (프레임워크 없음)

## 비기능 요구사항
- 외부 IP 접근 가능 (서버 불필요)
- 모바일/데스크탑 모두 동작
- 인터넷 연결 필요 (CDN 사용)

## 제외 범위 (2단계 이후)
- 드라이브 업로드
- 마스터 시트 연동
- 주차별 폴더 자동 생성
- 사원증 최종 디자인 합성
